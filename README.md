This script organizes your unorganized musci for you. teh final stucture looks like this:


~~~
Music --- <artist> --- <album> --- <song>
       |            |           |
       |            |           |- <song>
       |            |
       |            |- <album> --- <song>
       |	       	        |
       |	    		|- <song>
       |
       |- <artist> --- <album> --- <song>
                    |           |
                    |           |- <song>
                    |
                    |- <album> --- <song>
        	       	        |
        	    		|- <song>
~~~
